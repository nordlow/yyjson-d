# yyjson-d

[yyjson](https://github.com/ibireme/yyjson) D wrapper. When possible compliant
with Phobos's `std.json`.

Automatically provisions `libyyjson` locally via `provision.sh`.

Tested solely on Ubuntu 24.04.
